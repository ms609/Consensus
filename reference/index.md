# Package index

## Consensus methods

Functions for computing consensus trees from a set of input trees.

- [`Adams()`](https://constree.github.io/reference/Adams.md) : Adams
  consensus tree
- [`Average()`](https://constree.github.io/reference/Average.md) :
  Average consensus tree
- [`Frequency()`](https://constree.github.io/reference/Frequency.md) :
  Frequency-difference consensus tree
- [`Greedy()`](https://constree.github.io/reference/Greedy.md) : Greedy
  (extended majority-rule) consensus tree
- [`Local()`](https://constree.github.io/reference/Local.md) : Local
  consensus tree
- [`Loose()`](https://constree.github.io/reference/Loose.md) : Loose
  consensus tree
- [`Majority()`](https://constree.github.io/reference/Majority.md)
  [`MajorityRule()`](https://constree.github.io/reference/Majority.md)
  [`MR()`](https://constree.github.io/reference/Majority.md) :
  Majority-rule consensus tree
- [`MajorityPlus()`](https://constree.github.io/reference/MajorityPlus.md)
  : Majority-rule (+) consensus tree
- [`Quartet()`](https://constree.github.io/reference/Quartet.md) :
  Consensus tree minimizing quartet distance
- [`RStar()`](https://constree.github.io/reference/RStar.md) : R\*
  consensus tree
- [`Strict()`](https://constree.github.io/reference/Strict.md) : Strict
  consensus tree

## BHV tree space

Functions for computing distances and means in the
Billera–Holmes–Vogtmann tree space.

- [`BHVDistance()`](https://constree.github.io/reference/BHVDistance.md)
  [`BHVDist()`](https://constree.github.io/reference/BHVDistance.md)
  [`BHV()`](https://constree.github.io/reference/BHVDistance.md) :
  Geodesic (BHV) distance between trees
- [`BHVMean()`](https://constree.github.io/reference/BHVMean.md)
  [`BHVVariance()`](https://constree.github.io/reference/BHVMean.md) :
  Fréchet mean and variance in BHV treespace
